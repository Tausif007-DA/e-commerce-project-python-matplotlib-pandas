# e-commerce-project-python-matplotlib-pandas

# 📊 E-commerce Sales Performance Analysis

## 📌 Project Overview
This project analyzes an e-commerce dataset to understand business performance using Python and data visualization.

The goal is to move beyond basic charts and extract meaningful business insights.

---

## 📂 Dataset Details
The dataset contains 1000 rows with the following columns:

- Date  
- Region  
- Sales_Channel  
- Product  
- Category  
- Units_Sold  
- Price  
- Revenue  
- Cost  
- Profit  
- Marketing_Spend  
- Customer_Rating  
- Discount_Rate  

---

## 🛠 Tools & Technologies
- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook  

---

## 📊 Analysis Performed

### 1. Revenue Trend Analysis
- Monthly revenue trend using line chart  
- Identified fluctuations and growth patterns  

### 2. Category Performance
- Compared revenue across categories using bar chart  
- Identified top-performing categories  

### 3. Regional Analysis
- Compared revenue by region  
- Found high and low performing regions  

### 4. Marketing Effectiveness
- Scatter plot of Marketing Spend vs Revenue  
- Checked relationship between marketing and sales  

### 5. Profit Distribution
- Box plot used to detect outliers and variability  
- Analyzed consistency of profit  

---

## 🔍 Key Insights

- Revenue shows fluctuations with no consistent growth trend  
- Some categories contribute significantly more revenue  
- Regional performance is uneven  
- Weak relationship between marketing spend and revenue  
- Profit is highly variable with multiple outliers
